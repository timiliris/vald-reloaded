# VALD RELOADED — Twitch Live Notifier

![Chrome Web Store](https://img.shields.io/badge/Chrome-Extension-dc2626?style=for-the-badge&logo=googlechrome&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)

Ne rate plus jamais un live de Vald sur Twitch.

**Vald Reloaded** te previent instantanement quand Vald lance un stream, directement depuis ton navigateur.

![Preview](icons/icon128.png)

## Fonctionnalites

- **Notification instantanee** des qu'un live commence
- **Apercu du stream** : titre, jeu, viewers, duree
- **Historique complet** des streams passes
- **Clips Twitch** recents
- **Reseaux sociaux** de Vald (Twitch, YouTube, Spotify, Instagram, X...)
- **Badge en temps reel** avec le nombre de viewers
- **Personnalisable** : son, persistance, activation/desactivation des notifications

## Installation

### Chrome Web Store
> Lien bientot disponible

### Installation manuelle
1. Clone ce repo :
   ```bash
   git clone https://github.com/timiliris/vald-reloaded.git
   ```
2. Ouvre `chrome://extensions/` dans Chrome
3. Active le **Mode developpeur** (en haut a droite)
4. Clique sur **Charger l'extension non empaquetee**
5. Selectionne le dossier clone

## Permissions

| Permission | Utilisation |
|---|---|
| `alarms` | Verification periodique du statut du stream (toutes les 2 min) |
| `notifications` | Alertes desktop quand Vald est en live |
| `storage` | Sauvegarde locale de tes preferences |
